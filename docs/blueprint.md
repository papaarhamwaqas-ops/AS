# **App Name**: AS Track

## Core Features:

- Expense Tracking: Allow users to input and track daily expenses by category.
- Category Summary: Display summarized expense data for categories like food, transport, etc.
- Total Spending Display: Show the total daily spending in a prominent format.
- Detailed Expense Input: Capture expense name, amount, category, and description/notes.
- Landing Page with Hero Section: An engaging landing page before the main application with a striking hero section.

## Style Guidelines:

- Dominant background color: Black (#000000) for a high-end, premium feel.
- Primary color: Bold yellow (#FFC639) for buttons, highlights, and active states, creating a modern and vibrant look.
- Accent color: Dark gray (#333333) to provide subtle contrast against the black background, derived from the primary yellow.
- Headline font: 'Poppins', a geometric sans-serif for titles and primary text.
- Body font: 'Inter', a grotesque-style sans-serif for descriptions, labels and less prominent text. It will ensure excellent legibility on the dark background.
- Use simple, modern icons for expense categories and UI elements, with yellow (#FFC639) for active or highlighted states.
- Maintain clean spacing and a clear visual hierarchy, aligning to a minimalist, modern aesthetic typical of fintech apps.
- Incorporate subtle UI animations, such as soft shadows and glow effects, to enhance the app's premium feel.
